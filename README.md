# RoboLove 🤖

> "Find your perfect match... system optimized."

A robot dating app prototype built as a semester project.

## Features
- 🔄 Swipe through robot profiles one at a time (Tinder-style)
- 👤 View full robot profiles with image, description and interests
- ❤️ Initiate or decline sync (like/dislike)
- 📋 My Syncs list — view all your active connections
- 🔍 Filter synced robots by unit type
- 👾 My Profile page
- 💬 Chat interface (prototype)

## Built with
- HTML, CSS, JavaScript
- JSON data (robots.json)
- localStorage for saving syncs
- Mobile first design with media queries
- Orbitron font from Google Fonts
